Commande utile git:

git commit (dit si ton dossier est à jour)

git pull (fusionne le dossier master avec le tiens)

git add <chemin fichier/dossier> (permet de dire quel fichier il faut ajouter au master)

git commit -m "message" (associe un message au fichier ajoutées ex: version 2.0 de mon code)

git add -i (ou --interactive) (ouvre une fenetre de dialogue avec git qui permet d'ajouter facilement les nouveau fichier "add untracked")

git push origin master (envoie toutes les modification au master si ton dossier est à jour)

git rm -r --cached myFolder (Remove directory from git but NOT local)

git add -A (ajoute tout les modification au master)
